Welcome to "LLA" - Open Source Learning Program .

Learn Latvian in easy way by recognizing words and sounds.

HELP will guide you through the service. Follow these steps to learn.



Step 1. In window select the language you wish to learn : native - foreign



Step 2. In window select the topic you wish to learn and push button



Step 3. In menu tick the box : audio and/or text



Step 4. Start your lesson : chose the appropriate translation to the given foreign word.

Watch your score in



Step 5. In menu select :

save to save your result

reset to reset your result

exit to exit program
